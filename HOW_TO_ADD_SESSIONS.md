# How to Add & Manage Sessions

## Overview

Sessions are the core of your study planning in Prayatna. A session consists of tasks and subtasks that help you organize and track your study time effectively.

---

## Adding a New Session

### Method 1: From Sessions List

**[SCREENSHOT PLACEHOLDER: Sessions list page with "+" button highlighted]**

1. Navigate to the **Sessions** page
2. Click the **"+" button** (Add Session button) in the top-right corner
3. You'll be taken to the Create Session form

### Method 2: From Navigation Menu

1. Click the menu/hamburger icon
2. Select **"New Session"** or **"Create Session"**
3. You'll be taken to the Create Session form

---

## Creating a Session Step-by-Step

### Step 1: Enter Session Details

**[SCREENSHOT PLACEHOLDER: Session name input field]**

1. **Session Name** (Required)
   - Enter a descriptive name for your session
   - Examples: "Chapter 5 Study", "Math Practice Set", "Writing Assignment"
   - Avoid vague names; be specific about the content

2. **Session Duration** (Optional)
   - Set total time budget for the session
   - Enter hours, minutes, and seconds
   - Helps manage study time effectively
   - Leave empty for untimed sessions

### Step 2: Add Tasks

**[SCREENSHOT PLACEHOLDER: Task creation section with "+" button]**

1. Click the **"+ Add Task"** button below "Tasks in Session"
2. Task Modal opens with options:

   **Task Name** (Required)
   - Name of what you'll be studying/working on
   - Examples: "Reading Chapter 3", "Problem Set #1", "Essay Outline"

   **Has Subtasks?** (Yes/No)
   - **Select "No"** if this is a simple task with no breakdown
   - **Select "Yes"** if you want to divide this task into subtasks

   **Duration** (for tasks without subtasks)
   - Set time expected for this task
   - Used for time tracking and planning

3. Click **"Save Task"** to add the task

### Step 3: Add Subtasks (If Applicable)

**[SCREENSHOT PLACEHOLDER: Subtask creation modal]**

If you selected "Yes" for subtasks:

1. Click the **"+ Add Subtask"** button in the task modal
2. **Subtask Name** (Required)
   - Break down the task into smaller parts
   - Examples for "Essay Writing" task:
     - "Write Introduction"
     - "Write Body Paragraph 1"
     - "Write Conclusion"

3. **Duration** (Required)
   - Time allocated for this specific subtask
   - All subtask durations add up to task duration

4. Click **"Save"** to add subtask
5. Repeat for each subtask needed
6. Click **"Save Task"** to finalize the task with all subtasks

### Step 4: Add More Tasks (Optional)

**[SCREENSHOT PLACEHOLDER: Multiple tasks displayed as cards]**

1. Repeat Step 2 for each additional task
2. You can add as many tasks as needed
3. View all tasks as cards showing:
   - Task name
   - Task duration
   - Subtask list (if applicable)

4. To **edit a task**: Click the edit icon (pencil) on the task card
5. To **delete a task**: Click the delete icon (trash) on the task card

### Step 5: Save the Session

**[SCREENSHOT PLACEHOLDER: Save and Cancel buttons at bottom]**

1. Review all entered information
2. Ensure Session Name is filled
3. Click **"Save Session"** button
4. Session will be created and saved
5. You'll be redirected to the Sessions list

## Editing an Existing Session

### To Edit Your Own Session

**[SCREENSHOT PLACEHOLDER: Session view page with edit button]**

1. Navigate to Sessions list
2. Click the session to open it in view mode
3. Click the **Edit button** (pencil icon)
4. Make your changes:
   - Update session name
   - Modify tasks
   - Add/remove subtasks
5. Click **"Save Session"**
6. Changes are saved and you return to sessions list

### To Edit a System-Generated Session

**[SCREENSHOT PLACEHOLDER: System-generated session indicator]**

1. Open a system-generated session (marked with "System Generated" label)
2. Click **Edit**
3. A **confirmation modal** appears explaining:
   - "A copy will be created and saved to your account"
   - "The original template remains unchanged"
4. Click **"Continue Editing"**
5. The system creates a copy and opens it for editing
6. Make your changes and save
7. Your personalized copy is now saved; original template unchanged

## Duplicating a Session

### Quick Copy Method

**[SCREENSHOT PLACEHOLDER: Session card with menu options]**

1. From the Sessions list, find the session you want to copy
2. Click the **Duplicate button** (if available in session menu)
3. A copy is created with name: "Original Name (copy)"
4. The duplicate opens in edit mode (optional)
5. Modify as needed and save

### Manual Copy Method

1. Open the session you want to copy
2. Click **Edit**
3. Change the session name to something new
4. Make any modifications
5. Click **"Save Session"**
6. A new session is created; original remains unchanged

## Deleting a Session

### To Delete Your Own Session

**[SCREENSHOT PLACEHOLDER: Delete confirmation modal]**

1. Navigate to Sessions list
2. Right-click or long-press the session card
3. Select **Delete** from menu (or click delete icon)
4. **Confirmation modal** appears asking: "Are you sure?"
5. Click **"Delete"** to confirm
6. Session is removed from your list

### Guest Users
- Can delete **local-only sessions** (stored on device)
- Cannot delete **synced sessions** (require sign-in to manage)
- Cannot delete **system-generated sessions** (templates only)

### Signed-In Users
- Can delete any session you created
- Can hide system-generated sessions from your profile
- All deletions sync across devices

## Session Types Explained

### Your Sessions
- **Created by you** for your specific study needs
- **Fully editable** - change name, tasks, duration anytime
- **Deletable** - remove when no longer needed
- **Syncable** - saved to your account if signed in

### System-Generated Sessions
- **[SCREENSHOT PLACEHOLDER: System generated label indicator]**
- Created by Prayatna as study templates
- **Read-only** - cannot directly edit the original
- **Copiable** - create your own version when editing
- **Hideable** - remove from your profile (signed-in only)
- Label shows as "System Generated" next to session name

### Local Sessions (Guest Users)
- Created while not signed in
- Stored on device only
- Lost if browser cache is cleared
- Can be deleted anytime
- Not synced to any account

## Tips for Different Study Styles

### Focused Deep Work
- One main task per session
- Minimal subtasks
- Long duration (2-4 hours)
- Few distractions

### Structured Coursework
- Multiple related tasks
- Clear subtask breakdown
- Medium duration (1-2 hours)
- Organized by topic/chapter

### Practice & Drills
- Many short tasks
- No subtasks needed
- Varied durations
- Progress tracking emphasis

### Group Study
- Collaborative tasks
- Flexible subtasks
- Longer total duration
- Discussion checkpoints

## Troubleshooting Session Creation

### Session Won't Save
- Ensure **Session Name** is filled (Required field)
- Check all required fields are complete
- Look for error messages in red text
- Try again or refresh the page

### Tasks Aren't Showing
- Ensure tasks were added before saving
- Click "Save Task" button (not just closing modal)
- Tasks appear as cards after successful save

### Subtasks Disappeared
- Ensure "Has Subtasks" toggle was set to "Yes"
- Save each subtask individually
- Check that at least one subtask was added

### Can't Edit System-Generated Sessions
- This is intentional - system sessions are templates
- Click Edit to create and edit a personal copy
- Your copy will be independent and fully editable

### Session Lost After Browser Close
- **If signed in**: Sessions auto-save to account
- **If guest**: Sessions stored locally only
- **Solution**: Sign in to ensure sessions persist
